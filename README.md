# Spotify Song Popularity Prediction

## Project Overview
This project uses machine learning to predict Spotify song popularity using audio features.

## Dataset
Spotify Tracks Dataset (Kaggle)

## Platform
Google Colab

## Machine Learning Models
- Linear Regression
- Decision Tree
- KNN

## Evaluation Metrics
- MAE
- MSE

## Outputs
- Correlation Heatmap
- Scatter Plots
- Jitter Plot
- Line Plot
- Histogram
- Model Comparison

## How to Run
1. Open notebook in Google Colab
2. Upload dataset
3. Run all cells
